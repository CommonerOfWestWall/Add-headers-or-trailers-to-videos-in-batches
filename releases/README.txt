=====================================================
  视频片头片尾批处理工具 v4.4  —  使用说明
=====================================================

一、首次使用：放置 FFmpeg 内核（只需做一次）
------------------------------------------------------
本软件不内置 FFmpeg，需要你自行下载并和 exe 放在同一个文件夹。
好处：FFmpeg 有新版本时，直接替换 ffmpeg.exe / ffprobe.exe 即可升级，
      无需重新下载本软件。

【第一步：下载 FFmpeg】
  官方下载页：https://ffmpeg.org/download.html
  推荐用 gyan.dev 构建（Windows Essential 版即可）：
  https://www.gyan.dev/ffmpeg/builds/ffmpeg-release-essentials.zip

【第二步：解压，把两个文件复制过来】
  解压后找到 bin 文件夹里的：
    ✔ ffmpeg.exe
    ✔ ffprobe.exe
  把这两个文件复制到本软件 exe 所在目录。

  正确结果：
    📁 你的任意文件夹\
    ├── 视频片头片尾批处理工具.exe   ← 主程序
    ├── ffmpeg.exe                  ← ★ 从 FFmpeg 包里复制来的
    └── ffprobe.exe                 ← ★ 从 FFmpeg 包里复制来的

【第三步】双击 视频片头片尾批处理工具.exe 启动，搞定！

二、升级 FFmpeg 内核
------------------------------------------------------
  1. 前往官网下载最新版 FFmpeg
  2. 解压，用新的 ffmpeg.exe 和 ffprobe.exe 覆盖原来的两个文件
  3. 重新打开软件即生效

三、快速开始
------------------------------------------------------
  1. 在左侧"文件列表"添加要处理的视频文件或文件夹
  2. 选择片头 / 片尾文件，设置输出目录
  3. 点击"开始处理"即可

四、常见问题
------------------------------------------------------
Q: 弹出"缺少 FFmpeg"提示？
A: 把 ffmpeg.exe 和 ffprobe.exe 放到和本 exe 相同的文件夹里即可。

Q: 软件支持什么视频格式？
A: mp4 / mkv / avi / mov / flv / wmv / webm / m4v。

Q: 处理速度慢？
A: 切换"半专业调节"模式 → 硬件加速选 NVIDIA / AMD / Intel GPU。

Q: 第一次双击 exe 反应慢？
A: onefile 模式首次启动会解压到临时目录，稍等几秒是正常的。
